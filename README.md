## confluence

### Setup oh-my-zsh

#### Install confluence plugin
```zsh
pushd $ZSH/custom/plugins && \
  git clone git@github.com:johnlayton/martinslane.git confluence && \
  popd || echo "I'm broken"
```
```zsh
plugins=(... buildkite)
```

### Setup other

```zsh
pushd $HOME && \
  git clone git@github.com:johnlayton/martinslane.git .confluence && \
  popd || echo "I'm broken"
```

```zsh
source ~/.confluence/confluence.plugin.zsh
```
