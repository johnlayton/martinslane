## confluence

### Setup oh-my-zsh

#### Install confluence plugin
```zsh
pushd $ZSH/custom/plugins && \
    git clone git@github.com:johnlayton/martinslane.git martinslane && \
  popd || echo "I'm broken"
```
```zsh
plugins=(... martinslane)
```

### Setup other

```zsh
pushd $HOME && \
  git clone git@github.com:johnlayton/martinslane.git .martinslane && \
  popd || echo "I'm broken"
```

```zsh
source ~/.martinslane/martinslane.plugin.zsh
```
